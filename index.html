This document covers the installation, setup and use of this theme.

Requirements

You will need the following to use the theme:

  1. A fresh installation of WordPress.
  2. You have a username and password with an Administrator user role or FTP access.

Getting Started Back to Top

Locate your theme folder where you will find uoft.zip and u-of-t-ancillary-child-theme.zip

WordPress Theme and Child Theme Upload

  1. Login to your WordPress Admin (Ex: https://www.yourwebsite.com/wp-admin).
  2. Navigate to the Dashboard where you will find Appearance > Themes > Add New
  3. Upload Theme > Choose File .
  4. Choose uoft.zip and press the Install Now button.
  5. On the next screen, Click Go to Themes page

WordPress Child Theme Upload

  1. Upload Theme >Choose File.
  2. Choose u-of-t-ancillary-child-theme.zip and press the Install Now button.
  3. In the next screen, Click Activate.
  4. Your Themes are now installed.
If there happens to be any errors when you activate your Child Theme, notify your IT department or developers for troubleshooting. It's always best to send a screenshot and a quick explaination of what the error states.

FTP Upload

To manually upload your new WordPress theme, login with your credentials to your website and locate the wp-content folder in your WordPress install files. Upload the un-zipped 'uoft' folder and the 'u-of-t-ancillary-child-theme' folder into the: wp-content/themes folder.

Once uploaded, activate the theme by heading to the "Themes" menu in the WordPress Dashboard. Locate the U of T Child theme and Click Activate.

Install Recommended Plugins Back to Top

  1. When you first activate the theme, you will be notified of what plugins are recommended to use. Click Begin installing plugins.
  2. In the next screen, you are going to "Bulk Install" all the required plugins. Click the Dropdown > Select Install > Click the Checkbox next to "Plugin" > Click Apply
    3. Installation may take a few minutes, please be patient and do not refresh the page.
  3. After installation, scroll to the bottom of the page and click Return to Required Plugins Installer.
  4. In the next screen, you are going to "Bulk Activate" all the required plugins. Click the Dropdown > Select Install > Click the Checkbox next to "Plugin" > Click Apply
  5. Click Return to the Dashboard

Configurations Back to Top

Do the following configurations in order:

  1. Post Types Order
  2. Permalink Structure
  3. Media Upload Folder

Post Types Order

  1. Dashboard > Settings > Post Type Order
  2. From the Dashboard landing page, Click Settings Page

General:

  • Show / Hide re-order interface
  • Post > Show
  • Media > Show
  • Pattern > Show
  • Navigation Menu > Hide
  • News > Hide

Minimum Level to use this plugin:

  • Minimum Level to use this plugin > Administrator

Auto Sort:

  • Auto Sort > Uncheck

Admin Sort:

  • Admin Sort > Uncheck

Use query ASC / DESC parameter:

  • Use query ASC / DESC parameter > Uncheck

Archive Drag & Drop :

  • Post > Yes
  • Page > Yes
  • Media > Yes
  • Pattern > Yes
  • Navigation Menu > No
  • Taxonomies > No
  • Post Type > No
  • Field Group > No
  • News > No

Next / Previous Apply Uncheck

  • Click Save Settings > Uncheck
  1. Click Save Settings

Permalink Structure

  1. Dashboard > Settings > Permalinks
  2. Click Custom Structure > Uncheck the tags currently in place: 
    %year%
    %monthnum%
    %day% 
    %postname%
  3. Check the tags: 
    /%category%/%postname%/
  4. Scroll to the bottom of the page and Save Changes

Media Upload Folder

  1. Dashboard > Settings > Media
    • Uploading Files:
    • Uncheck > Organize my uploads into month- and year-based folders
  2. Scroll to the bottom of the page and Save Changes

Import Demo ContentBack to Top

Work off of U of T demo content located in your Theme Folder. The XML file is included within the download package with the file name uoft-demo-data.xml

  1. Dashboard > Tools > Import
  2. WordPress > Run Importer
  3. Choose File > uoft-demo-data.xml > Upload file and import
  4. In the next screen,
    Assign Authors > Choose "or assign posts to an existing user" and choose the user.
  5. Import Attachments > Check Download and import file attachments and Click Submit.

    The import will take a few minutes (5 - 10 minutes).

    Please be patient and do not refresh the page.

    To verify the import is working, take a look at the favicon in your browser tab to see if it is spinning as an indication it is downloading.

    <- Success! It's importing!

Homepage and Navigations Back to Top

Set Homepage

  1. Dashboard > Settings > Reading
  2. Your homepage displays > Select "A static page" > From the Homepage: drop down, Select Home
  3. Scroll to the bottom and Save Changes.

Set Homepage Navigation and Footer Navigation

  1. Dashboard > Appearance > Menus

Click the Manage Locations tab and match the Theme Location to the Assigned Menu from the drop downs.

Manage Locations:

  • Primary > primary
  • Footer Column 1 > Footer Column 1
  • Footer Column 2 > Footer Col 2
  • Footer Column 3 > Footer Column 3
  1. Save Changes

Theme Options Back to Top

With Theme Options you can add:

  • -> Homepage logo
  • -> Footer logo
  • -> Homepage Header Alert Bar
  • -> Homepage Header Button
  • -> Footer Social Media Icons

Homepage Logo

  1. Dashboard > Theme Options > Header > Header Logo > Add Image
  2. Publish > Update

Footer Logo

  1. Dashboard > Theme Options > Footer > Footer Logo > Add Image
  2. Publish > Update

Homepage Header Alert Bar

  1. Dashboard > Theme Options > Header > Show Alert Bar >Show
  2. Alert Bar >Add Image
  3. Media Library > Search "alert" > Choose your alert icon > Click Select
  4. Alert Content > Add content
  5. Alert Color > Select a Colour > Enter a HEX Colour [ Brand Colours ] > Publish > Update

    NOTE: Use the Brand Colours HEX Colors.    Example: U of T Blue 
    #1E3765
    6. Check to see if Secondary Colours pass Web Accessiblity Normal Text WCAG AA: https://webaim.org/resources/contrastchecker/

Homepage Header Button

  1. Dashboard > Theme Options > Header > Header Button > Select Link
  2. Publish > Update

Footer Social Media Icons

  1. Dashboard > Theme Options > Company Information > Social Media
  2. Paste in Social URLs
  3. Publish > Update
TURN OFF Auxiliary Department Required

Turn off the Requirement to have ALL four social URLs

Dashboard > ACF > Theme Options > Edit > Auxiliary Department > Edit > Validation > Required (off)
